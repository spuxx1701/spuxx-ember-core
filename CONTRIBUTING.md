# How To Contribute

## Installation

- `git clone <repository-url>`
- `cd spuxx-ember-core`
- `npm install`

## Linting

- `npm run lint`
- `npm run lint:fix`

## Building the addon

- `cd spuxx-ember-core`
- `npm build`
